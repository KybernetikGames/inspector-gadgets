Inspector Gadgets is a Unity plugin which enhances various aspects of the Unity Editor to streamline your development workflow.

Download it for free and check out the Documentation to get started.

Transform Inspector

Local/World space, Copy/Paste/Snap, Freeze Child Transforms, and more.

Hierarchy

See components, tags, layers, and child counts of each object in the Hierarchy.

Persist After Play Mode

Change things in Play Mode and keep the changes when you return to Edit Mode.

Drag & Drop

Advanced options for handling files dropped into Unity such as overwriting existing assets instead of creating new ones.

Decorator Attributes

Various attributes allow you to customise the way fields are drawn in the Inspector and even add things like buttons.

Object Reference Fields

Quick find button, see the referenced object's Inspector, enhanced Drag & Drop support, and more.

Context Menus

Dozens of useful functions are added to the context menu (right click menu) of each field in the Inspector based on its type.

Auto Hide UI

UI objects are hidden when you aren't working with them and the camera will focus on them when you select one.

Missing Scripts

Find missing scripts in all scenes and assets in your project so you can go through and fix them.
